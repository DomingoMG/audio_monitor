# Roadmap

## Near Term

- Harden macOS monitoring across more hardware combinations
- Improve diagnostics for permission and routing failures
- Refine monitor mute and volume behavior

## Medium Term

- Add direct selected-output routing on macOS
- Expand test coverage around native failure cases
- Improve example app diagnostics for real device testing

## Longer Term

- Add a Windows backend aligned with native listen-to-device behavior
- Re-evaluate Linux support without widening plugin scope
