# Changelog

## 0.0.1

- Initial Flutter desktop plugin structure for `audio_monitor`
- Typed Dart API for device listing and monitoring control
- macOS monitoring implementation for:
  `selected input -> current system default output`
- Mute and unmute support for the monitor path
- Dedicated monitor volume control, including configuration before `start()`
- Example app for manual testing
- README and GitHub Pages documentation scaffolding
- MIT license for the initial public release
